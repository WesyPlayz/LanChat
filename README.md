# LanChat

## Creator
[ Ryan L Harding ]( https://github.com/WesyPlayz )

## Overview 
Want to talk uninterrupted by internet or service providers? LanChat helps by allowing devices of the same network to connect and send messages.

## Why Create LanChat?
LanChat is designed to be used for any confidential, important, work-related, or other use cases which do not require internet or a service provider. Users may only need locally connected data to be stored and accessed.

## Who is This Designed For?
LanChat is designed percisely for people in work environments who need a fast and easy way to discuss with no extra distractions within the app itself and only need it locally within the workplace
